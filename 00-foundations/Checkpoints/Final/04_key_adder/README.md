# Key Adder

### Total Points: 6.00

### Description:



Create a Key Adder that loops over an Object's keys, finds(filters) all number values, and returns
the sum of all the numbers.

Example:

```
// INPUT: {a0:0, a1:1, a2:2, a3:3, a4:4}
keyAdder.call({a0:0, a1:1, a2:2, a3:3, a4:4})
// OUTPUT: 10;

```
